# 概览

-   全局API —— 全局会用到的API
-   组合式API —— vue3所拥有的组合式API
-   选项式API —— vue2所拥有的选项式API
-   内置内容 —— 指令、组件、特殊元素和特殊属性
-   单文件组件 —— 语法定义、
-   进阶API —— 渲染函数、服务端渲染、TS工具类型和自定义渲染
